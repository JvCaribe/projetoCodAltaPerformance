Nosso projeto consiste em um catálogo de filmes. Na página inicial são exibidas informações sobre os filmes em uma tabela. A aplicação permite que os usuários pesquisem o filme específico desejado e, caso não encontrem, eles têm a possibilidade de adicionar o filme. Os títulos dos filmes são clicáveis, redirecionando o usuário à página do filme no IMdb.

Os dados dos filmes são obtidos através da API OMDb, permitindo o preenchimento automático de informações como título, gênero, diretor, duração, poster. 

O sistema utiliza o framework Django e também HTML, CSS, JavaScript para construção da interface e da lógica da aplicação. A persistência dos dados da aplicação é realizada através do banco de dados SQLite, integrado ao Django.
